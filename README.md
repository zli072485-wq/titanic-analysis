# 泰坦尼克号生存分析

## 项目目的
学习 pandas 数据处理和 matplotlib 可视化，分析泰坦尼克号乘客生存率的影响因素。

## 数据集
来自 Kaggle 的泰坦尼克号训练集，包含891名乘客的信息（年龄、性别、舱位、是否幸存等）。

## 分析步骤

### 1. 数据清洗
- `Age` 列（177个缺失值）：用中位数填充
- `Cabin` 列（687个缺失值）：缺失太多，直接删除
- `Embarked` 列（2个缺失值）：用众数（S）填充

### 2. 特征工程
- 新增 `Sex_Num` 列：male→0，female→1
- 新增 `AgeGroup` 列：<18为 Child，≥60为 Elder，其余为 Adult

### 3. 分组分析
按船舱等级（Pclass）、性别、年龄分组，计算每组的平均生存率。

### 4. 可视化
画出不同舱位和性别的生存率对比柱状图。

## 主要发现
- **一等舱女性**存活率最高（约96.8%）
- **三等舱男性**存活率最低（约13.5%）
- **儿童**存活率普遍高于成人
- **女性**存活率显著高于男性
